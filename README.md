# «Заметки» — PWA-приложение для деловых встреч (Кейс-задача №4, React + TypeScript)

Приложение для создания, редактирования, удаления и поиска заметок во время совещаний,
встреч и конференций. React + TypeScript, сборка через Vite. Работает как Progressive Web
App — устанавливается на телефон/компьютер без магазина приложений, поддерживает офлайн-режим.

## Технологический стек

React 18 + TypeScript (strict mode) + Vite. Хранение: типизированный слой IndexedDB
(`src/db.ts`) и File System Access API (`src/fileStorage.ts`, типы в `src/fs-access.d.ts`)
с запасным вариантом скачивания файла для браузеров без этого API.

## Разработка

```bash
npm install
npm run dev        # http://localhost:5176, hot reload
```

## Сборка для продакшена / хостинга

```bash
npm run build       # соберёт статические файлы в dist/
npm run preview     # локальный просмотр собранной версии
```


## Структура проекта

```
notes-app-react/
  index.html
  public/                    — манифест PWA, service worker, иконки (копируются как есть)
  src/
    main.tsx                 — точка входа, монтирование React-дерева
    App.tsx                  — корневой компонент: состояние, обработчики CRUD и хранения
    components/
      NoteCard.tsx            — карточка заметки в списке
      NoteModal.tsx            — форма создания/редактирования заметки
      StorageBar.tsx           — переключатель "База данных / Файл на устройстве"
    db.ts                     — типизированный слой доступа к IndexedDB
    fileStorage.ts             — открытие/сохранение JSON-файла с заметками
    types.ts                  — типы Note, StorageMode
    fs-access.d.ts             — ambient-декларации File System Access API
    css.d.ts                   — декларация для импорта CSS
    style.css
```

## Функциональность и способы хранения — без изменений

Функциональность:
создание/редактирование/удаление/поиск заметок, переключение между базой данных (IndexedDB)
и файлом на устройстве, офлайн-режим и установка на iOS/Android. Подробности и скриншоты —
в «руководство_пользователя_заметки.docx», переданном вместе с отчётом (интерфейс визуально
не изменился, экраны из руководства актуальны).
